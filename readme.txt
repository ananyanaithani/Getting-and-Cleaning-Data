#Getting and Cleaning Data - Course Project 

* Download and unzip the data from "https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip" to your working directory
* In the R environment, set the working directory as desired and source "run_analysis.R"
* Use the file "averages_dataset.txt" to read the final tidy dataset
